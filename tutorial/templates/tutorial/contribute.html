To begin with, there are some background knowledge you need to know:

  • What is pythonanywhere?

    • Our website is developed basing on pythonanywhere webserver. After logging into the account, you will be led to Dashboard. This Dashboard is the background control for our website. Click “WEB” tab and you will see our website’s name: bluedevilresistome.pythonanywhere.com. Anything you change from the background control will be reflected on this website. Just regard it as a “virtual” computer! You can install software, create folders and files from Bash Consoles as on your local computer. On this virtual computer, we installed Django and developed website.


  • What is Django framework?

    • Our web is based on Django framework. A basic understanding on how Django works would be very helpful for a beginner. More information can be found on Django tutorial, there you will learn how to use Django to write first app.


  • How to use Github?

    • GitHub is a powerful tool to record and update our codes. Instead of directly editing code from Bash Consoles in pythonanywhere, we usually download the complete repository from GitHub to our own computer for local test. Our pythonanywhere account mentioned above has been linked to GitHub account. You can clone the repository to Bash Console if anything new added, and push the updated repository to GitHub if you do change something from Bash Consoles. GitHub provides detailed tutorials on how to use git command from terminal.


  • How to use virtual machine (VM) for local development?

    • For local test, if you don’t want to install too many new tools on your local computer, a virtual machine can be helpful. Duke university provides students with one VM container on vcm.duke.edu.
